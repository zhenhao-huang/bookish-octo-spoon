# 文件检测工具

### 检测缺失文件和损坏文件

### 该工具可以用来检测有序排列中的缺失文件和因损坏文件大小变得很小的文件，文件格式是exr（如有需求请修改代码中的"exr"为其他格式）

用的数据库是MongoDB数据库，如果没有可以到https://www.mongodb.com/download-center/community 下载，或者注释掉GUIRun.py中PyMongo所有数据插入语句

test里的数据可以用来测试，拖入或者打开都可以
